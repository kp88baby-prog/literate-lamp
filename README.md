# Autonomous AI Gold Trading Agent

## Overview
This repository contains an AI-powered autonomous gold (XAU/USD) day trading agent. The agent is designed to use real-time market data, make AI-driven trading decisions, and interface with the OANDA trading API. It includes a web-based frontend for monitoring and manual trade control.

## Features
- Python-based backend for trading logic
- FastAPI server for RESTful endpoints
- Web dashboard for monitoring prices, reviewing trades, and manual control
- Supports multiple trading strategies
- Implements risk management principles
- Example files provided; customize as needed.

## File Structure
- `main_trading_agent.py`: Core orchestration script
- `api_server.py`: REST API interface
- `modules/`: Logic modules for data feed, AI decisions, and trading execution
- `web/`: Frontend HTML, CSS, and JavaScript files
- `.env.example`: Template for secret environment variables, such as OANDA API tokens

More files and logic will be added incrementally to meet the full specifications.
